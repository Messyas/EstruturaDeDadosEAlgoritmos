### O que é a notação BIG O?

- É uma forma de medir eficiencia de um algoritmo a partir da quantidade de ciclos necessarios para se realizar uma operacao.
Essa notacao foi criada para entender como um algoritmo se comporta a medicao que o tamanho do input aumenta.

Por exemplo, Existem varias formas de se descrever um algoritimo de fibonacci, caso nao saiba, a seria de fibonacci é data por Fn = Fn-1 + Fn-2, ou seja, a soma dos dois antecessores gera o proximo numero da sequancia, no caso seria possivel escrever esse algoritimo usando recursao (funcao que chama a si mesma), iterativo (for, while etc) e um metodo fechado matematicamente,  se estiver com dificuldade, entender o n como uma posicao de um array talvez ajude a entender a ideia (É so uma alucao, mas se tiver dificil de entender leia este artigo: https://medium.com/@marcellguilherme/fibonacci-9aded34b2fb6).
